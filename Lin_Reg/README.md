Задача: прогнозирование стоимости квартир

Методы:

 ---- Исследование распределения признаков и целевой переменной, поиск выбросов, анализ корреляций.

 ---- Предобработка: Обработка пропусков, кодирование категориальных признаков, масштабирование.

 ---- Моделирование: Реализация и сравнение линейных моделей с разными типами регуляризации (L1, L2, ElasticNet).

 ---- Оптимизация: GridSearchCV для подбора оптимальных гиперпараметров моделей.

 ---- Валидация: Сравнение моделей по метрикам RMSE, MAE, R²

Стек: Python, pandas, scikit-learn, matplotlib, seaborn.
