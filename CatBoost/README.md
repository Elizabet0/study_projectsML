Классификация клиентов банка для прогнозирования депозита

Задача: Построить модель бинарной классификации для прогноза подпишется ли клиент на депозит

Методы:

 ---- Первичный анализ и очистка данных, обработка пропусков, преобразование категориальных признаков (LabelEncoder/One-Hot Encoding), разделение данных (80/20).

 ---- Решающие деревья: Построила, визуализировала и проанализировала дерево. Исследовала влияние глубины (2-10) на переобучение.

 ---- Случайный лес: Обучен ансамбль из 100 деревьев. Производительность и устойчивость к переобучению сравнены с лучшим одиночным деревом.

 ---- Градиентный бустинг CatBoost: влияние learning_rate (0.01, 0.1, 0.3) на качество модели.
