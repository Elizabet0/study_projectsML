Обнаружение мошеннических операций с кредитными картами


Задача: Разработать модель для бинарной классификации транзакций (мошеннические/легитимные) на основе анонимизированного исторического датасета.

Решение:

 ---- Предобработка: Масштабирование признаков Time и Amount, стратифицированное разделение на выборки.

 ---- Моделирование: Реализован стек из 6 моделей — от базовых (kNN, дерево) до ансамблевых (Random Forest, Stacking).

 ---- Оценка: F1-score и Recall, анализ через confusion matrix и ROC-AUC.

 ---- Оптимизация: Подбор гиперпараметров с помощью GridSearchCV.

Вывод: Наилучший баланс между выявлением фрода (Recall) и минимизацией ложных тревог (Precision) показала модель Random Forest.
