# Учебные проекты по MLке

Здесь собраны мои проекты, выполненные в ходе обучения! :)

## Список проектов:
1. Обнаружение мошеннических операций — бинарная классификация на несбалансированных данных, сравнение алгоритмов и работа с дисбалансом классов.

2. Прогнозирование подписки на депозит — анализ и сравнение моделей (решающие деревья, случайный лес, градиентный бустинг).

3. Влияние предобработки данных — оценка методов обработки пропусков, выбросов и масштабирования на качество модели.

4. Исследование KNN — применение алгоритма k-ближайших соседей для задач классификации и регрессии, анализ влияния гиперпараметра k.

5. Полный цикл анализа кредитных карт — от предобработки до подбора гиперпараметров, включая оценку через F1-score и ROC-AUC.
